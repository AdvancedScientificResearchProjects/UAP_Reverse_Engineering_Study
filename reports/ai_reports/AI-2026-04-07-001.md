### LLM Visual Analysis (Claude Opus 4.6) / Визуальный анализ LLM

#### EN

**Method:** Multimodal LLM (Claude Opus 4.6) received the original and cropped artifact image without project context or ECP data. The prompt included 6 analysis axes:

1. Physical description — shape, size, color, texture
2. Surface patterns — ridges, grooves, symmetry
3. Material hypothesis — visual hypotheses about the object's nature
4. Comparison candidates — specific analogues (species, taxa, mineral types)
5. Anomalous features — what doesn't fit standard categories
6. Confidence assessment — confidence per hypothesis

The model had no access to ECP reports, reference data, or CV pipeline results. Fully blind analysis.

#### RU

**Метод:** Мультимодальная LLM (Claude Opus 4.6) получила оригинальное и кадрированное изображение артефакта без контекста проекта и ECP-данных. Промпт включал 6 осей анализа:

1. Physical description — форма, размер, цвет, текстура
2. Surface patterns — гребни, борозды, симметрия
3. Material hypothesis — визуальные гипотезы о природе объекта
4. Comparison candidates — конкретные аналоги (виды, таксоны, типы минералов)
5. Anomalous features — что не вписывается в стандартные категории
6. Confidence assessment — уверенность по каждой гипотезе

Модель не имела доступа к ECP-отчётам, референсным данным или результатам CV-пайплайна. Анализ полностью слепой.

### Interpretation / Интерпретация

## Limitations / Ограничения

### EN

- All material hypotheses are visual-only (RGB)
- Chemical composition requires: hyperspectral camera, OES, XRF, Raman, SEM-EDS
- Anomaly detection limited by single-image input (no training distribution)
- Depth is relative, not metric
- Single viewpoint limits 3D reconstruction

### RU

- Все гипотезы о материале основаны только на визуальных данных (RGB)
- Определение химического состава требует: гиперспектральной камеры, OES, XRF, рамановской спектроскопии, SEM-EDS
- Обнаружение аномалий ограничено одним изображением (нет обучающей выборки)
- Глубина относительная, не метрическая
- Единственная точка съёмки ограничивает 3D-реконструкцию
