## Purpose / Назначение

**EN:** Session-organization tool for batch HSP scanning across the Track 7 prime / services-prime / DOE-consortium-operator constellation. The roster surfaces public-source candidates so that scan sessions can be coordinated without re-discovering names per session. Roster entries pair a public **role-class** with a public **employer-of-record** and the canonical **public-source URL** (corporate About page + LinkedIn URL).

**RU:** Инструмент организации пакетных HSP-сессий сканирования по созвездию prime / сервисных prime / операторов-консорциумов DOE из Трека 7. Реестр выводит публично-источниковых кандидатов так, чтобы сессии сканирования можно было координировать без повторного нахождения имён каждый раз. Запись в реестре связывает публичный **ролевой класс** с публичным **работодателем-по-записи** и каноническим **URL публичного источника** (страница «About» + LinkedIn-URL).
