## 4. PROTOCOL NOTES / ЗАМЕТКИ ПО ПРОТОКОЛУ

### Methodological Notes / Методологические замечания

1. **Data contamination / Контаминация данных**: Ekaterina and Tatyana heard Vladislav before responding. These responses cannot be treated as independent observations. Future sessions require strictly separate questioning. / Екатерина и Татьяна слышали Владислава до своих ответов. Эти ответы нельзя трактовать как независимые наблюдения. Для будущих сеансов нужен строго раздельный опрос.

2. **Prior exposure / Предварительный контакт**: Vladislav had seen the object previously and received minimal context ("found in the steppe"). Not a fully blind test. / Влад видел объект ранее и получил предварительную информацию ("нашли в степи"). Не полностью слепой тест.

3. **No control object / Отсутствие контрольного объекта**: No ordinary stone or part was presented for calibration. / Не предъявлялся обычный камень/деталь для калибровки.

4. **Detail variation / Различия в детализации**: Vladislav — 10 min active scanning, detailed stream. Ekaterina — multiple sessions (text + call), most detailed interpretation. Tatyana — brief but with unique data (ageless, military civilization, crab/octopus). / Влад — 10 минут активного сканирования, детальный поток. Екатерина — множественные сеансы, самая детальная интерпретация. Татьяна — краткая, но с уникальными данными.

5. **Protocol v2.0 strength / Сильная сторона протокола v2.0**: Separation into free description + structured questions works — free part captured metaphors and images, structured part extracted details. / Разделение на свободное описание + структурированные вопросы работает — свободная часть дала метафоры и образы, структурированная вытянула детали.
