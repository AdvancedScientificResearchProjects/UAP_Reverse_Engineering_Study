## 1. OVERVIEW / ОБЗОР

### EN

This report presents the validation results of Extended Cognitive Perception (ECP) session ECP-2026-04-04, in which three ECP percipients assessed an unidentified artifact (UAP-FRAG-001) using the standardized ASRP ECP Protocol v2.0.

After all percipient assessments were collected, the Program Director disclosed the verified reference data about the object. This report compares percipient responses against the reference data to evaluate accuracy, response agreement, and protocol effectiveness.

## 5. KEY FINDINGS / КЛЮЧЕВЫЕ ВЫВОДЫ

### EN

1. **Core function correctly identified.** All three responses described the object as a signal/beacon-type device. This is the primary functional characteristic confirmed by the reference data.

2. **System membership repeated across all recorded responses.** All three recorded responses stated the object is part of a larger system, not a standalone artifact. This matches the reference description of a multi-element triangulation installation.

3. **No consciousness — repeated across all recorded responses.** All three recorded responses reported the absence of own consciousness in the object, describing it as a tool/instrument. This is consistent with the reference characterization.

4. **Non-terrestrial origin — unanimous in the recorded responses.** All three responses described the object as non-terrestrial. This is consistent with the reference data.

5. **Terrain correctly identified by one percipient.** Vladislav described "mountains, desert, sunset in red-burgundy tones" — matching the Tian Shan mountain range location in Kazakhstan.

6. **Temporal aspect detected indirectly.** Tatyana stated the object is "ageless, no linear time" — an indirect but notable alignment with the reference data indicating temporal navigation purpose.

7. **Material partially identified.** All three described mineral/non-metallic properties. The reference states "biometal" — a more complex category that none identified precisely, though all detected the non-standard material nature.

8. **Radioactivity not detected by any percipient.** The variable radioactivity property was not perceived. This may represent a limitation of cognitive perception for physical measurement properties.

9. **One incorrect assertion.** Vladislav perceived dynamic flight/movement associated with the object. The reference states the object is stationary and was never airborne. This may represent analytical overlay (interpretation) rather than raw perception — Vladislav himself marked this as an interpretation.

10. **Courier/transporter correctly identified.** Both percipients who assessed the person in the photograph correctly identified him as a passive carrier, not an operator or owner.

### RU

1. **Основная функция определена верно.** Во всех трёх ответах объект описан как сигнальное/маяковое устройство. Это основная функциональная характеристика, подтверждённая референсными данными.

2. **Принадлежность к системе повторяется во всех зафиксированных ответах.** Во всех трёх ответах объект описан как часть бо́льшей системы. Это совпадает с описанием многоэлементной триангуляционной установки.

3. **Отсутствие сознания повторяется во всех зафиксированных ответах.** Во всех трёх ответах объект описан как инструмент без собственного сознания.

4. **Внеземное происхождение — повторяющаяся тема во всех ответах.** Во всех трёх ответах объект описан как внеземной.

5. **Местность корректно определена одним перципиентом.** Владислав описал "горы, пустыню, закат в красно-багровых тонах" — совпадение с районом Тянь-Шаньского хребта в Казахстане.

6. **Темпоральный аспект обнаружен косвенно.** Татьяна указала, что объект "безвозрастной, нет линейного времени" — косвенное, но примечательное совпадение с навигационным назначением во времени.

7. **Материал определён частично.** Все трое описали минеральные/неметаллические свойства. Референс указывает "биометалл" — более сложная категория, которую никто не идентифицировал точно, хотя все обнаружили нестандартную природу материала.

8. **Радиоактивность не обнаружена.** Свойство переменной радиоактивности не было воспринято ни одним перципиентом. Это может представлять ограничение когнитивного восприятия для физических измеряемых свойств.

9. **Одно некорректное утверждение.** Владислав воспринял динамику полёта. Референс указывает, что объект стационарный. Это может быть аналитическим наложением (интерпретацией) — Владислав сам отметил это как интерпретацию.

10. **Курьер/перевозчик определён корректно.** Оба перципиента, оценивавших фотографию человека, верно идентифицировали его как пассивного переносчика, а не оператора или владельца.

## 6. PROTOCOL EFFECTIVENESS / ЭФФЕКТИВНОСТЬ ПРОТОКОЛА

| Aspect / Аспект | Assessment / Оценка |
|-----------------|-------------------|
| **Free description → structured questions flow** | Effective — free section captured metaphors and raw signal, structured questions extracted details / Эффективно — свободная часть уловила метафоры и сырой сигнал, структурированные вопросы извлекли детали |
| **Signal vs interpretation separation** | Partially effective — Vladislav used it well, others less consistently / Частично эффективно — Владислав использовал хорошо, остальные менее последовательно |
| **Blinding** | Compromised — sequential questioning in group call. Future sessions must be conducted individually / Нарушено — последовательный опрос в групповом звонке. Будущие сеансы должны проводиться индивидуально |
| **Self-assessment utility** | Useful — Vladislav's self-identified "interpretation" of flight dynamics correlated with the one incorrect assertion / Полезно — самоидентифицированная Владиславом "интерпретация" динамики полёта коррелировала с единственным некорректным утверждением |

### Recommendations for future sessions / Рекомендации

1. **Individual sessions only** — no group calls, no exposure to other percipient data
2. **Include control objects** — ordinary stones/artifacts mixed in to calibrate
3. **Record session timing** — exact start/end for all percipients
4. **Formalize pre-session state** for all percipients (Ekaterina's was not recorded)

## 7. CONCLUSION / ЗАКЛЮЧЕНИЕ

### EN

The ECP session ECP-2026-04-04 showed clear response-level agreement on several key object characteristics. The core function (navigation beacon), system membership, absence of consciousness, and non-terrestrial origin appeared across all three recorded responses. The descriptive accuracy summary was 70% exact matches and 90% exact-or-partial matches against the disclosed reference data.

The primary methodological limitation was compromised blinding due to sequential group questioning. Because of that limitation and the very small sample size, these results should be treated as preliminary descriptive findings. Within those limits, the session supports using the ECP approach as a structured hypothesis-generation tool within the ASRP research framework, not as standalone validation.

### RU

Сеанс КП ECP-2026-04-04 показал заметную согласованность ответов по нескольким ключевым характеристикам объекта. Основная функция (навигационный маяк), принадлежность к системе, отсутствие сознания и внеземное происхождение повторились во всех трёх зафиксированных ответах. Описательный итог по точности составил 70% точных совпадений и 90% точных или частичных совпадений относительно раскрытых референсных данных.

Основным методологическим ограничением было нарушение ослепления из-за последовательного группового опроса. С учётом этого ограничения и крайне малого числа участников результаты следует трактовать как предварительные описательные наблюдения. В этих пределах сеанс поддерживает использование подхода КП как структурированного инструмента генерации гипотез в рамках исследовательской программы ASRP, а не как самостоятельной валидации.
