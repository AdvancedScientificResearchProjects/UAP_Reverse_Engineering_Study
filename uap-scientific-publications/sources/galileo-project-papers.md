**EN:** The Galileo Project at Harvard's Center for Astrophysics was founded in 2021 to systematically search for technological signatures of non-human-origin objects in Earth's vicinity using purpose-built ground-based observatories. The project's anchor papers describe the multi-modal sensor stack (visible / infrared / radio / audio / magnetometer), data-pipeline calibration, and machine-learning classification stages; subsequent technical reports describe the first deployed observatory at Harvard and the candidate-event statistics from initial operations. The Galileo Project is referenced across the post-2017 UAP physics literature as the most-formally-instrumented academic-institutional entry into the observation-program layer of the corpus.

**RU:** Galileo Project в Harvard Center for Astrophysics был основан в 2021 г. для систематического поиска технологических сигнатур объектов нечеловеческого происхождения в окрестностях Земли с использованием специально построенных наземных обсерваторий. Опорные работы проекта описывают мультимодальный сенсорный стек (видимый / инфракрасный / радио / аудио / магнетометр), калибровку конвейера данных и стадии классификации машинного обучения; последующие технические отчёты описывают первую развёрнутую обсерваторию в Гарварде и статистику кандидат-событий начальных операций. Galileo Project ссылается в литературе по физике UAP пост-2017 как наиболее формально-инструментированный академически-институциональный вход в наблюдательно-программный слой корпуса.
