## Замечания о надёжности источников / Notes on Source Reliability

**RU:**
- **royallib.com** — протокол-относительные URL (`//royallib.com/get/...`), нужно добавить `https:` для использования
- **coollib.net** — `/b/ID.../download` даёт FB2; `/download_new` даёт исправленную версию
- **avidreaders.ru** — страницы-шлюзы загрузки, а не прямые ссылки на файлы; требуется один редирект
- **phantastike.com** — подтверждена прямая отдача файлов (ZIP/DOC/PDF)
- **naturalworld.guru** — ссылки `/book/download/ID`, формат не маркирован на странице (вероятно PDF/DOC)
- **flibusta.club** — заблокирована в России; доступна через VPN или зеркало (flibusta.is, flibusta.site)
- **archive.org** — только отсканированные PDF, без нативных FB2/EPUB
- **rulit.me** — шлюз загрузки `/get-BOOKID.html`; форматы подтверждены, но закрыты сессией

**EN:**
- **royallib.com** — protocol-relative URLs (`//royallib.com/get/...`), prepend `https:` to use
- **coollib.net** — `/b/ID.../download` gives FB2; `/download_new` gives corrected version
- **avidreaders.ru** — download gateway pages, not raw file links; require one redirect step
- **phantastike.com** — direct file serving confirmed (ZIP/DOC/PDF)
- **naturalworld.guru** — `/book/download/ID` links, format not labeled on page (likely PDF/DOC)
- **flibusta.club** — blocked in Russia; accessible via VPN or mirror (flibusta.is, flibusta.site)
- **archive.org** — scanned PDFs only, no FB2/EPUB native
- **rulit.me** — download gateway at `/get-BOOKID.html`; formats confirmed but gated by session
