Bilingual extraction from two Vadim Chernobrov (Вадим Чернобров, 1965–2017) video-lectures
that concentrate heavily on the Time Machine / Машина Времени (MV).
Focus: technical statements by Chernobrov himself. Audience banter, UFO anecdotes unrelated
to the device, and philosophical digressions trimmed.

**Sources:**
- **HRON-14** — File 07 (`07_MashinaVremeni_Hronoput_3h04.txt`), Гузерипль (Gузерипль),
  Республика Адыгея, гостевой дом «Лесной-3», экспедиция «Космопоиск», 28 июня 2014 г.
  (YouTube `9k-mlSfU9V0`, 3h04m).
- **MUR-14-13** — File 13 (`13_Hronoput_Murom_2h37.txt`), Муром, Владимирская область,
  транзитом по пути с Северного Ледовитого океана на Медведицкую гряду, середина лета
  2014 г. (YouTube `yeL7msYWhxc`, 2h37m).

**IMPORTANT — DUPLICATION WARNING.** File 13 (MUR-14-13) and File 01 (`01_Chernobrov_MashinaVremeni_2h37.txt`,
already analysed as **MUROM-14-01**) share duration (2h37m), city (Муром) and dating
